# 📚 دليل العرب في تركيا - المنصة الشاملة

> منصة معلومات قانونية وحياتية شاملة للجالية العربية في تركيا، خاصة السوريين. توفر استشارات قانونية، معاملات إدارية، وأدوات حسابية ذكية.

🇹🇷 **اللغة:** العربية (RTL) | 🔐 **الأمان:** TypeScript Strict Mode | ⚡ **الأداء:** Next.js Static Generation

---

## ✨ المميزات الرئيسية

### 🎯 المحتوى
- **680+ مقالة قانونية** موثقة ومحدثة بشكل دوري
- **40+ خدمة مأجورة** مع تفاصيل كاملة
- **15+ سيناريو استشاري** مع تقييم المخاطر (Safe/Medium/High/Critical)
- **نماذج قانونية جاهزة** (عقود إيجار، اعتراضات، إلخ)
- **أكواد أمنية** موثقة وسهلة البحث

### 🛠️ الأدوات التفاعلية
- **المستشار الذكي**: يجيب على الأسئلة القانونية المعقدة
- **حاسبة منع الدخول**: حساب فترة المنع بناءً على البيانات
- **البحث العام الذكي**: بحث سريع على كل المحتوى
- **خريطة تفاعلية**: مواقع المؤسسات الحكومية
- **مشاركة سهلة**: WhatsApp و Social Media

### 📱 التصميم
- ✅ **دعم كامل للعربية RTL**
- ✅ **استجابي بنسبة 100%** (Mobile/Tablet/Desktop)
- ✅ **رسوميات متحركة سلسة** (Framer Motion)
- ✅ **تتبع التقدم** في المقالات (localStorage)
- ✅ **واجهة عصرية** مع ألوان تركية الطابع

## 🚀 البدء السريع

### المتطلبات
- Node.js 18+ 
- npm أو yarn أو pnpm

### التثبيت والتشغيل

```bash
# 1. استنساخ المستودع
git clone <repo-url>
cd daleel-arab-turkiye

# 2. تثبيت المكتبات
npm install

# 3. إنشاء ملف البيئة (اختياري - للإنتاج)
cp .env.example .env.local

# 4. تشغيل خادم التطوير
npm run dev

# 5. افتح المتصفح
open http://localhost:3000
```

## 📁 بنية المشروع

```
src/
├── app/                    # صفحات Next.js (File-based routing)
│   ├── page.tsx           # الصفحة الرئيسية
│   ├── layout.tsx         # التخطيط الأساسي
│   ├── article/[id]/      # صفحات المقالات الديناميكية
│   ├── consultant/        # المستشار الذكي
│   ├── services/          # الخدمات المأجورة
│   ├── codes/             # دليل الأكواد الأمنية
│   └── [other routes]/    # صفحات أخرى
│
├── components/            # مكونات React القابلة لإعادة الاستخدام
│   ├── Navbar.tsx        # شريط التنقل
│   ├── GlobalSearch.tsx   # البحث العام الذكي
│   ├── ArticleView.tsx    # عرض المقالات
│   ├── LeafletMap.tsx     # الخريطة التفاعلية
│   ├── Footer.tsx         # التذييل
│   └── [other components]/
│
└── lib/                   # دوال وبيانات مشتركة
    ├── data.ts           # الإعدادات والملاحات
    ├── articles.ts       # قاعدة بيانات المقالات (680+ سطر)
    ├── consultant-data.ts # السيناريوهات القانونية
    ├── dictionary.ts     # القاموس القانوني
    ├── codes.ts          # الأكواد الأمنية
    └── [other data files]/
```

## 🛡️ الأمان والخصوصية

- ✅ **No API calls to external services** - كل البيانات محلية
- ✅ **TypeScript Strict Mode** - فحص صارم للأنواع
- ✅ **Environment variables** - لا تخزن أسرار في الكود
- ⚠️ **إخلاء مسؤولية:** المعلومات للأغراض التعريفية فقط، لا تغني عن الاستشارة القانونية الرسمية
